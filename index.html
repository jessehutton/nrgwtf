My first major project that I undertook as a Junior Web Developer was a Status Dashboard that accepted trouble tickets, logged them for specific site services in a SQL Database and reported the current status of services based off of the IP that was viewing the page as well as a logged in account option for off-site monitoring. General website and payment status information was available to the general public but in order to view more detailed service status information, a login or accessing the website from an internal IP was required. This was all authenticated using PHP and displayed using Angular 1.5. After 5 years of bulding, redesigning and maintaining all of the websites for the company, my final project was implementing the new payment and booking system across all websites including converting all Angular over to Vue.js.
